## 🚀 Overview

**Planik AI** is a state-of-the-art SaaS platform designed to bridge the gap between clients needing architectural solutions and contractors looking for projects. Leveraging the power of Generative AI, Planik AI allows clients to generate beautiful, highly accurate floor plans instantly, post them as projects, and receive competitive bids from verified contractors. 

Built with scalability, security, and world-class UX in mind, this project serves as a comprehensive ecosystem for the modern construction and architectural space.

---

## ✨ Key Features

- **🤖 AI Floor Plan Generation**: Clients can describe their dream home or building, and our AI engine instantly generates optimal, proportional, and highly detailed floor plans.
- **🔐 Role-Based Access Control (RBAC)**: Secure, distinct environments tailored for Clients, Contractors, and Administrators using Supabase Auth.
- **⚡ Real-Time Bidding System**: Contractors can place bids on active projects and map out timelines directly through the platform.
- **💬 Integrated Messaging**: Secure, real-time communication between clients and contractors to finalize details, share updates, and collaborate.
- **📊 Real-Time Progress Tracking**: Contractors update milestones securely, keeping clients fully informed throughout the project lifecycle. 
- **🌙 Premium UI/UX & Dark Mode**: Built with Tailwind CSS utilizing a "Glassmorphism" aesthetic, sleek micro-animations, and full dark/light mode support.

## 🎯 Dashboards Breakdown

Planik AI provides distinct, feature-rich dashboards specially crafted for 3 different types of users:

### 👤 1. Client Dashboard
The command center for users looking to build or renovate.
* **AI Generation Studio**: The core tool where clients input parameters (room counts, dimensions, style) to generate AI floor plans.
* **Project Management**: Create, view, and manage property development projects. 
* **Bid Review System**: Review incoming contractor bids for their projects, compare quotes, timelines, and contractor ratings to select the winning bid.
* **Active Project Tracking**: Monitor ongoing assigned projects with real-time percentage progress updates and milestone tracking.
* **Conversations Hub**: Direct, real-time messaging with assigned contractors to easily discuss modifications or track progress.
* **Review & Ratings**: Leave verified feedback and ratings upon project completion to ensure quality standards across the platform.

### 👷 2. Contractor Dashboard
A comprehensive workspace for professionals to find work and manage clients.
* **Project Marketplace**: Browse a dynamic feed of open projects and AI-generated floor plans awaiting construction/execution. 
* **Bid Management Engine**: Submit new bids (specifying cost estimates and timelines), and track the status of pending/accepted/rejected bids.
* **Active Assignments**: View all projects they have won, including client details and associated AI floor plans.
* **Milestone & Progress Delivery**: Update project completion percentages securely so clients are kept actively in the loop. 
* **Client Chat**: Chat instantly with clients to negotiate terms, request clarifications, or provide fast updates.

### 🛡️ 3. Admin Dashboard
The ultimate control panel to manage the ecosystem, ensure security, and track growth.
* **Advanced Analytics**: At-a-glance metrics covering platform revenue, active projects, user growth, and AI utilization.
* **User Management**: View, disable, remove, or modify accounts for both Clients and Contractors. 
* **Project Oversight**: Monitor the status of every single project flowing through the platform; resolve disputes effectively.
* **Audit Logs**: A rich, immutable log of all crucial actions (logins, project creations, deletions, bid modifications) for maximum security and traceability.
